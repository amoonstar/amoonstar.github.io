1 python中深拷贝,浅拷贝的区别并举例出来

在 Python 中，深拷贝和浅拷贝都是用于复制可变对象的方式。

浅拷贝（Shallow Copy）只是复制了可变对象的引用，而不是对象本身，因此原始对象和副本对象共享相同的子对象。当对原始对象或副本对象中的子对象进行修改时，另一个对象也会发生变化。浅拷贝可以使用 copy() 方法实现。

深拷贝（Deep Copy）则是复制了可变对象及其所有子对象，因此原始对象和副本对象不共享任何子对象。当对原始对象或副本对象中的子对象进行修改时，另一个对象不会发生变化。深拷贝可以使用 deepcopy() 方法实现。
